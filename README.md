## Installation

### Prerequisites

- Python 3.x
- Pandas library
- Matplotlib library
- Colorama library
- Pyfiglet library

### Steps
1. Clone the repository:
   ```bash
   git clone https://github.com/Jastreas/Python_Pandas_Matplotlib_Final_Test.git
   ```
2. Open the folder:
    cd Python_Pandas_Matplotlib_Final_Test
3. Run the code from each exercise:
    py [number].py
    **Example:**
    py 5.py
